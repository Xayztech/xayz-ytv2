Tutorial Lengkap

Panduan step-by-step cara menggunakan XAYZ YT API Platform

01
Dapatkan API Key
Kunjungi halaman API Key, masukkan nama kamu (opsional), lalu klik tombol "Generate API Key Gratis!". API key akan muncul — simpan key tersebut karena tidak bisa dipulihkan jika hilang.
🔑 Generate Key
02
Test API di Browser
Buka URL berikut di browser (ganti KEY dengan API key kamu):
https://youtubeonline-dl.vercel.app/xayz/yt-machine/api/search?q=test&apikey=XAYZ-XXXX-XXXX-XXXX

Jika berhasil, kamu akan melihat response JSON dengan daftar video.

03
Download MP3 / MP4
Gunakan endpoint ytmp3 atau ytmp4 dengan URL video YouTube dan pilih kualitas yang diinginkan:
# MP3 320kbps /xayz/yt-machine/api/ytmp3?url=YT_URL&quality=320&apikey=KEY # MP4 1080p /xayz/yt-machine/api/ytmp4?url=YT_URL&quality=1080&apikey=KEY
04
Integrasikan ke Kode Kamu
API ini mendukung Node.js, Python, PHP, JavaScript, Bash, dan bahasa apapun yang bisa melakukan HTTP request. Lihat contoh kode lengkap di halaman Docs.
// Node.js CJS - Install package langsung npm install xayz-yt const { ytmp3 } = require('xayz-yt'); const result = await ytmp3('https://youtu.be/dQw4w9WgXcQ', 320); console.log(result.download.url);
06
Pantau Penggunaan
Kunjungi halaman Cek API Key untuk melihat statistik penggunaan: total request, IP yang mengakses, domain, user-agent, dan riwayat request terbaru.
📊 Cek Penggunaan
