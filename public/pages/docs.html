XAYZ YT API Docs

REST API untuk YouTube — Search, Download MP3/MP4, Metadata, Channel. Gratis, unlimited, free lifetime.

✓ Free Lifetime ✓ Unlimited ✓ REST API ✓ AI Sandbox Ready

API ini mendukung semua HTTP method (GET, POST), semua format response (JSON), cross-origin (CORS enabled), dan mendukung sandbox/headless environments untuk AI agents. Semua request memerlukan API Key.

Authentication

Sertakan API key di setiap request menggunakan salah satu cara berikut:

// Header (Recommended) x-api-key: XAYZ-XXXX-XXXX-XXXX // Query Parameter ?apikey=XAYZ-XXXX-XXXX-XXXX // Atau ?key=XAYZ-XXXX-XXXX-XXXX

➡ Dapatkan API Key Gratis

Base URL

https://youtubeonline-dl.vercel.app/xayz/yt-machine/api

Ganti youtubeonline-dl.vercel.app dengan domain deployment kamu.

Search

GET/search?q=KEYWORD
POST/search
ParameterTypeWajibDeskripsi
qstringYaKata kunci pencarian. Alias: query, keyword, k
apikeystringYaAPI Key. Bisa juga via header x-api-key

YT MP3 (Audio)

GET/ytmp3?url=YT_URL&quality=128
ParameterTypeWajibDeskripsi
urlstringYaURL YouTube video. Alias: link, u
qualitynumber—Kualitas audio (kbps): 92, 128, 256, 320. Default: 128
apikeystringYaAPI Key
GET /xayz/yt-machine/api/ytmp3?url=https://youtu.be/dQw4w9WgXcQ&quality=320&apikey=KEY // Response { "status": true, "metadata": { "title": "...", "thumbnail": "...", ... }, "download": { "status": true, "quality": "320kbps", "availableQuality": [92, 128, 256, 320], "url": "https://cdn.../file.mp3", "filename": "Rick Astley - Never Gonna Give You Up (320kbps).mp3" } }

YT MP4 (Video)

GET/ytmp4?url=YT_URL&quality=360
ParameterTypeWajibDeskripsi
urlstringYaURL YouTube video
qualitynumber—Resolusi: 144, 360, 480, 720, 1080. Default: 360
apikeystringYaAPI Key

Metadata

GET/metadata?url=YT_URL

Mengambil metadata lengkap video: judul, thumbnail, channel, statistik, deskripsi, tags, dan lainnya.

Channel

GET/channel?input=@handle
ParameterTypeWajibDeskripsi
inputstringYaHandle (@nama), URL channel YouTube, atau channel ID

Contoh — Node.js (CJS)

const axios = require('axios'); const API_KEY = 'XAYZ-XXXX-XXXX-XXXX'; const BASE = 'https://youtubeonline-dl.vercel.app/xayz/yt-machine/api'; // Search async function search(keyword) { const { data } = await axios.get(`${BASE}/search`, { params: { q: keyword }, headers: { 'x-api-key': API_KEY } }); return data; } // Download MP3 async function downloadMp3(url, quality = 128) { const { data } = await axios.get(`${BASE}/ytmp3`, { params: { url, quality }, headers: { 'x-api-key': API_KEY } }); return data.download?.url; } // Download MP4 async function downloadMp4(url, quality = 720) { const { data } = await axios.get(`${BASE}/ytmp4`, { params: { url, quality }, headers: { 'x-api-key': API_KEY } }); return data.download?.url; } (async () => { const results = await search('Rick Astley'); console.log(results.results[0].title); const mp3 = await downloadMp3('https://youtu.be/dQw4w9WgXcQ', 320); console.log('MP3 URL:', mp3); })();

Contoh — PHP

<?php $API_KEY = 'XAYZ-XXXX-XXXX-XXXX'; $BASE = 'https://youtubeonline-dl.vercel.app/xayz/yt-machine/api'; function xayz_request($endpoint, $params = []) { global $API_KEY, $BASE; $url = "$BASE/$endpoint?" . http_build_query($params); $ctx = stream_context_create(['http' => [ 'method' => 'GET', 'header' => "x-api-key: $API_KEY\r\n" ]]); return json_decode(file_get_contents($url, false, $ctx), true); } $search = xayz_request('search', ['q' => 'Rick Astley']); echo $search['results'][0]['title']; $mp3 = xayz_request('ytmp3', ['url' => 'https://youtu.be/dQw4w9WgXcQ', 'quality' => 320]); echo $mp3['download']['url']; ?>

Contoh — JavaScript Fetch (Browser)

const API_KEY = 'XAYZ-XXXX-XXXX-XXXX'; const BASE = 'https://youtubeonline-dl.vercel.app/xayz/yt-machine/api'; async function xayz(endpoint, params = {}) { const url = new URL(`${BASE}/${endpoint}`); Object.entries(params).forEach(([k,v]) => url.searchParams.set(k, v)); const res = await fetch(url, { headers: { 'x-api-key': API_KEY } }); return res.json(); } // Search const s = await xayz('search', { q: 'lofi hip hop' }); console.log(s.results); // Download const dl = await xayz('ytmp3', { url: 'https://youtu.be/dQw4w9WgXcQ', quality: 128 }); window.open(dl.download.url); // Open download link

Gunakan sebagai NPM Library

// Install npm install xayz-yt // Usage (CJS) const { search, ytmp3, ytmp4, metadata, channel } = require('xayz-yt'); // Search video const results = await search('lofi hip hop'); // Download MP3 const mp3 = await ytmp3('https://youtu.be/dQw4w9WgXcQ', 320); console.log(mp3.download.url); // Download MP4 const mp4 = await ytmp4('https://youtu.be/dQw4w9WgXcQ', 1080); console.log(mp4.download.url); // Get metadata const meta = await metadata('https://youtu.be/dQw4w9WgXcQ'); console.log(meta.title, meta.statistics); // Get channel info const ch = await channel('@RickAstleyYT'); console.log(ch.title, ch.statistics.subscriber);
